MiniMax Algorithm
	1. Start with the current game state.
	2. Check if the current node is a terminal state or the maximum search depth is reached.
	3. If it is a terminal state, return its evaluation score.
	4. If it is the Maximizing Player's turn:
		a. Initialize the best value as negative infinity.
		b. Recursively evaluate all child nodes.
		c. Select the maximum evaluation value.
	5. If it is the Minimizing Player's turn:
		a. Initialize the best value as positive infinity.
		b. Recursively evaluate all child nodes.
		c. Select the minimum evaluation value.
	6. Return the best value to the parent node.
	7. Choose the move with the optimal evaluation score.
